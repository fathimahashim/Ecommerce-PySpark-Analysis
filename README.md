
# 🛒 E-Commerce Data Analysis using PySpark

## 📌 Project Overview
This project analyzes an online retail dataset using PySpark in Google Colab. The goal is to extract meaningful business insights from large-scale transactional data.

---

## 🚀 Technologies Used
- Python
- PySpark
- Google Colab
- Matplotlib

---

## 📊 Key Features
- Data cleaning and preprocessing
- Handling missing and invalid values
- Sales trend analysis
- Customer behavior analysis
- Country-wise revenue analysis

---

## 📈 Analysis Performed
- Top selling products
- Top customers
- Monthly sales trends
- Revenue distribution by country
